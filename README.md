# rod
Custom webview implementation for bun based on tauri libraries

# Required system dependencies
This package uses wry from the tauri project to render webviews. wry requires the following packages:

## Windows
Edge webview runtime installed

## Linux
See the [official readme](https://github.com/tauri-apps/wry/tree/dev?tab=readme-ov-file#linux-dependencies)
For tray see [this readme](https://github.com/tauri-apps/tray-icon?tab=readme-ov-file#dependencies-linux-only)
